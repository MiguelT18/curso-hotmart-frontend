¿QUÉ VAS A LOGRAR?

Iniciarte en la programación web de la forma correcta. Se han preparado una serie de clases teóricas y prácticas que te permitirán entender y aplicar los conceptos básicos y avanzados de HTML5. El curso inicia en un nivel de complejidad bastante básico para aquellos que están empezando en el mundo de la programación y termina con explicaciones sobre CSS3, para ir aprendiendo a otorgarle formato o belleza a nuestras páginas.

Te sentirás totalmente preparado y animado al finalizar el curso y podrás aplicar estos conceptos en tus proyectos personales y profesionales. El contenido del curso está totalmente actualizado con la última versión de los programas que vas a utilizar.

¿CÓMO LO VAS A LOGRAR?

Es importante que puedas seguir cada una de las clases, pausando entre ellas las mismas paulatinamente para que puedas practicar de manera secuencial la ejercitación desarrollada, bajo una secuencia de hilo conductor, donde desde un nivel cero podrás ir aprendiendo las bases de HTML5 y CSS3, hasta llegar a la última clase, donde podrás darte cuenta que tienes un producto terminado, el cual lo podrás usar de referencia o de guía para ir emprendiendo sobre tus proyectos personales.

Módulo de CSS Grid

CSS Grid Layout permite crear rápidamente cuadrículas sencillas y potentes de forma prácticamente instantánea. También se destaca por permitir dividir una página en áreas o regiones principales, considerando la relación en términos de tamaño, posición, sin dejar de lado la posibilidad de permitir alinear elementos en filas y columnas.

Módulo de Bootstrap-5

Bootstrap-5 es un framework front-end utilizado para desarrollar aplicaciones web y sitios mobil first, o sea, con un layout que se adapta a la pantalla del dispositivo utilizado por el usuario. En este bono aprenderás los fundamentos de esta tecnología y cómo utilizarla, para ello crearás una Landing Page, bajo la técnica Responsive Design, esta técnica se usa en la actualidad para tener una misma web adaptada a los diferentes dispositivos: Computadora, Tablet o Smartphone.
